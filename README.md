# 🔐 AI Secure Data Intelligence Platform

AI-powered web application that detects sensitive data like passwords, API keys, tokens, and emails from text or files, assigns a risk score, and masks critical information.

---

## 🚀 Features
- 🔍 Sensitive data detection (Email, Password, API Keys, Tokens)
- ⚠️ Risk classification (Critical / High / Medium / Low)
- 🔒 Data masking for security
- 📊 Interactive dashboard (charts & analytics)
- 📄 Downloadable PDF report
- 🕒 Scan history tracking

---

## 🛠️ Tech Stack
- Backend: Python (FastAPI)
- Frontend: HTML, CSS, JavaScript
- Server: Uvicorn

---

cd backend
